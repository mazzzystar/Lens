Lens is a Chrome extension that provides instant, sharp AI insights on any text you select while browsing social media or any webpage.

## Installation

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode" (toggle in top right)
3. Click "Load unpacked"
4. Select the `Lens` folder

## Setup

1. Click the Lens icon in your Chrome toolbar
2. Select your AI provider (OpenAI or Anthropic)
3. Enter your API key
4. Click "Save Settings"

## Usage

1. Select any text on a webpage
2. Right-click and choose "Lens: See Through This"
3. Get instant insight
