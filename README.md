## Installation

### Quick Install

1. [Download the latest release](https://github.com/mazzzystar/Lens/releases/download/v0.0.3/Lens-v0.0.3.zip)
2. Unzip the file
3. Open Chrome â†’ `chrome://extensions/`
4. Enable **Developer mode** (top right)
5. Click **Load unpacked** and select the unzipped folder

### From Source

1. Clone this repository:
   ```bash
   git clone https://github.com/mazzzystar/Lens.git
   ```
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode** (top right)
4. Click **Load unpacked** and select the `Lens` folder

## Setup

1. Go to [OpenRouter](https://openrouter.ai/settings/keys) and create an API key
2. Click the Lens icon in Chrome toolbar
3. Paste your key (Lens is open source, so your key stays safe)
4. Set your preferred response language
5. Done

Default model is Gemini 3 Pro. You can switch to Claude 4 Opus, but Gemini tends to provide deeper thinking.

## Why I Built This

I often come across interesting takes online and want AI to analyze them deeply. Every time, I had to:
1. Open Gemini
2. Copy-paste the text
3. Type the same prompt: "What do you think? Give me deep insights"

I wanted a tool to shorten this path so I could browse and think more frequently.

Also: **the internet rewards extreme opinions**. Engagement-driven algorithms amplify the loudest voices, not the most thoughtful ones. We need to browse with our brains turned on.

Lens helps you see past the noise. When you encounter someone's hot take, let AI examine whether the reasoning holds up â€” or reveal the deeper dynamics at play.
